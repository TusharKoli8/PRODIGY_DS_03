 **Decision Tree Classifier – Bank Marketing Prediction**


Built a Decision Tree Classifier to predict whether a bank customer will subscribe to a term deposit based on demographic and campaign-related data.

**What I Did**

Performed data cleaning and preprocessing
Explored the dataset using EDA
Encoded categorical features
Trained and optimized a Decision Tree model
Evaluated performance using accuracy, ROC-AUC, and cross-validation

**Tech Stack**

Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

**Results**

Baseline Accuracy: 51.16%
Tuned Accuracy: 58.56%
Best Model: Entropy, Max Depth = 3

**Files**

decision_tree_classifier.py → Main model code
bank.csv → Dataset
outputs/ → Visualizations and results

**Key Insight**

Call duration and previous campaign success were the strongest factors influencing customer subscription.
